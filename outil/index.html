Bienvenue dans l'Outil ADP 2026

Lancer de Poids DI-POI · CRMEF Inezgane · EPS

👋

Bienvenue !

Cet outil vous accompagne dans l'aide à la décision pédagogique pour le cycle de Lancer de Poids. Il analyse les données de votre classe et génère automatiquement un plan de cycle différencié.

🔒 Vos données restent confidentielles sur votre appareil. Rien n'est envoyé sur Internet. L'outil fonctionne aussi hors connexion.
📋

① Diagnostic de classe

Saisissez les données de vos élèves : nom, 3 observables et performance. Vous pouvez importer un fichier CSV depuis le terrain.

A = 3/10 B = 6/10 C = 9/10 ou valeur 0–10
💡 Cliquer sur un élève dans le tableau ouvre sa fiche détaillée.
🎯

② Axes prioritaires

L'outil analyse les lacunes collectives et classe automatiquement les axes pédagogiques par pertinence. Sélectionnez 3 ou 4 axes.

📊 Le % affiché = part d'élèves en Profil A sur l'observable lié.
📅

③ ④ Paramétrage & Planification

Choisissez le nombre de séances (6 à 10). L'outil génère un tableau différencié A/B/C séance par séance.

🔄 Réévaluez après la séquence du milieu.
⚙️

Fonctionnalités disponibles

📚 Multi-classes — gérez plusieurs classes sans ressaisir
📱 Mode tablette — boutons A/B/C pour saisie terrain
💾 Export/Import JSON — sauvegarde et transfert complet
🗃 Données exemple — tester immédiatement sans ressaisir
