[3] TÂCHE — 4 ÉTAPES SÉQUENTIELLES


ÉTAPE 1 — PRODUCTION DES CODES
Depuis grille_v1 + fichier Excel joint :
  · Produire 1 code comportemental par degré par observable
  · Format : 2 caractères MAXIMUM · lisible à 50cm sur fiche imprimée
  · Code dominant = comportement le plus représentatif du degré
  · Codes distincts pour A · B · C de chaque observable
  · Convention couleurs obligatoire :
    Degré A → fond #FCEBEB (rouge clair) · Arial Bold ≥ 14pt
    Degré B → fond #FAEEDA (amber clair) · Arial Bold ≥ 14pt
    Degré C → fond #EAF3DE (vert clair)  · Arial Bold ≥ 14pt


ÉTAPE 2 — ⚑ PAUSE INTERACTIVE ORIENTÉE
Après avoir produit le tableau des codes · STOP.
Présenter la recommandation de saisie selon [FAMILLE_APS] :


  [Si Sports collectifs :]
  "Pour votre APS ([APS]) · nous recommandons :
   → ENTOURER le code dominant (A · B · C) pour les observables
     de type Niveau à valider
   → ÉCRIRE le nombre pour les observables de type Fréquence
   Durée maximale recommandée : 2 secondes par élève.
   Ordre de saisie : séquentiel élève 1 → élève N.
   Confirmez-vous cette procédure ou signalez une contrainte terrain ?"


  [Si Athlétisme :]
  "Pour votre APS ([APS]) · nous recommandons :
   → ÉCRIRE le chrono au format STRICT : 10"2 · 1'30" · 0,90
     (secondes → 10"2 · minutes → 1'30" · mètres → 0,90)
   → puis ENTOURER le degré correspondant (A · B · C)
   Durée maximale recommandée : 5 secondes par passage.
   Ordre de saisie : ordre de passage.
   Confirmez-vous ou signalez une contrainte terrain ?"


  [Si Gymnastique :]
  "Pour votre APS ([APS]) · nous recommandons :
   → ENTOURER le degré (A · B · C) pour les observables Niveau
   → ÉCRIRE le nombre pour les observables Fréquence
   Durée maximale recommandée : 5 secondes par passage.
   Observation : pendant la phase d'enchaînement uniquement.
   Confirmez-vous ou signalez une contrainte terrain ?"


  [Si Sports de raquette :]
  "Pour votre APS ([APS]) · nous recommandons :
   → ENTOURER le code dominant après chaque échange clé
   Durée maximale recommandée : 3 secondes par observation.
   Fréquence : 1 observation tous les 3 échanges minimum.
   Confirmez-vous ou signalez une contrainte terrain ?"


  [Si Sports de combat :]
  "Pour votre APS ([APS]) · nous recommandons :
   → ENTOURER le degré dominant pendant l'assaut
   Durée maximale recommandée : 3 secondes par élève.
   Observation : en bord de tapis · position latérale.
   Confirmez-vous ou signalez une contrainte terrain ?"


ÉTAPE 3 — INTÉGRATION DES CONFIRMATIONS

  · Produire la procédure normée finalisée
  · Si contrainte signalée → adapter la procédure
